Proyecto Aplicación mobile Bruno Cánepa

En este proyecto se realiza una aplicacion estática que muestre shows y restaurantes, en la que se pueda navegar por distintas paginas.
La aplicación fue hecha con React Native, utilizando Expo para ejecutarla.

La aplicación se llama "Culinarte", haciendo referencia al arte culinario (cocina y musica).

Luego de haber clonado el repositorio se debe ejecutar el siguiente comando:

"npx expo start"

Una vez ejecutado el comando se deberá escanear el código QR que aparece desde la aplicación de expo si es android o desde la camara si es iOS.
